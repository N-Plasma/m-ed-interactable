Welcome to a web version of the python section of my project "Multiplatform Encryption / Decryption"
This is a example of what can be done with the project (aswell as a way for people who may be unable to run python to try this)

This is using PyScript
The Actual project is [here](https://codeberg.org/N_Plasma/Multiplatform-Encrypt_Decrypt)
This sites Github repo is [here](https://github.com/N-Plasma/m-ed-interactable)

Note : If using direct key mode theres some other features
1. When using the key "0" it will give the ascii codes of the input without the seperator character
2. When using the key "1" it will give the output without multiplying the characters (not a unique feature, just unintended effect)
